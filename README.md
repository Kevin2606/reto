## Funcionamiento
1. Clone el repositorio.
2. Use el comando ```composer update```
3. Uso mediante un cliente RestApi como "ThunderClient"
- La url de la Api es: ```http://localhost/ApolT01-017/reto/```
- Metodos GET, POST, UPDATE, DELETE.
